# Lista de Tarefas React

Aplicação desenvolvida como projeto prático de Frontend utilizando React e Styled Components.

## Funcionalidades

* Adicionar tarefas
* Marcar tarefas como concluídas
* Remover tarefas
* Filtrar tarefas:

  * Todas
  * Ativas
  * Concluídas
* Alternar entre tema claro e escuro
* Interface responsiva e estilizada com Styled Components

## Tecnologias Utilizadas

* React
* Vite
* Styled Components
* JavaScript (ES6+)

## Como Executar

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta do projeto:

```bash
cd lista-tarefas
```

3. Instale as dependências:

```bash
npm install
```

4. Execute a aplicação:

```bash
npm run dev
```

5. Abra o navegador no endereço informado pelo Vite.
